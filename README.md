Meu objetivo não é apenas escrever código, mas sim **entender negócios e criar soluções completas de ponta a ponta**. Unindo tecnologia, comunicação e estratégia, busco me tornar um Arquiteto de Software capaz de desenhar e implementar sistemas seguros, escaláveis e que resolvem problemas reais.

---

### 🏆 Projeto em Destaque: EsteticaFlow Ultra 4.0
> **Status:** 🟢 Em Produção (Sistema Proprietário / Código Fechado)
> 
> Arquitetura e desenvolvimento "do zero" de uma SPA Mobile-First focada em gestão de alto padrão para clínicas de estética premium. Muito além de um CRUD genérico, o sistema resolve dores complexas de agendamento, segurança e retenção de clientes.
>
> **Stack Principal:** React 19, TypeScript 5.8, Tailwind CSS 3, Vite 6, Supabase (PostgreSQL, Auth, RLS).
> **Engenharia:** Git Flow estrito e Versionamento Semântico (SemVer) com changelogs dinâmicos.
> 
> **Destaques da Arquitetura (Regras de Negócio Implementadas):**
> * **Smart Agenda:** Lógica pesada de validação de dias/horários, bloqueios da gerência em tempo real e "Smart Filters" para categorias médicas.
> * **CRM e IA Insights:** Módulo autônomo de retenção de pacientes que identifica inatividade (30+ dias) e aciona fluxos de recuperação via WhatsApp com parâmetros dinâmicos.
> * **RBAC e Audit Logs Estritos:** Controle de acesso por rotas e registro imutável de operações invisíveis (quem fez, o que fez, quando fez), essencial para auditoria de nível enterprise.
> * **Painel Administrativo Dinâmico:** Gestão de variáveis globais persistidas no banco, permitindo edição de templates de remarketing interpolados em tempo real na interface (ex: `{clientName}`, `{procedure}`).
